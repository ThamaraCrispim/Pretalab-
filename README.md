# Formação em Análise de Dados – PretaLab (Ciclo 15)

Este repositório reúne meus estudos, exercícios e projetos desenvolvidos durante a formação em Análise de Dados promovida pela PretaLab.

A PretaLab é uma iniciativa que capacita mulheres negras e indígenas para atuação na área de tecnologia, com foco em desenvolvimento técnico, empregabilidade e inclusão no mercado.

---

##  Objetivo da Formação

Desenvolver competências técnicas e analíticas essenciais para atuação como Analista de Dados, incluindo:

- Manipulação e análise de dados com Python
- Utilização de SQL para consulta e integração de dados
- Limpeza, transformação e preparação de dados
- Análise estatística descritiva
- Visualização e interpretação de dados
- Desenvolvimento de projetos aplicados ao mundo real

---

## Competências Desenvolvidas

### 🐍 Python para Análise de Dados
- Leitura e escrita de arquivos (CSV, JSON, TXT)
- Manipulação de dados
- Uso de pacotes e módulos
- Integração com banco de dados

### 🗄️ Banco de Dados e SQL
- Conceitos fundamentais de banco de dados
- Operações CRUD
- Consultas SQL (SELECT, filtros e JOIN)
- Integração entre Python e SQLite

### 📊 Análise de Dados com Pandas e NumPy
- Limpeza de dados
- Tratamento de valores ausentes
- Filtragem e transformação
- Preparação de dados para análise

### 📈 Estatística Aplicada
- Média
- Mediana
- Moda
- Variância
- Desvio padrão
- Distribuição de dados
- Interpretação estatística

### 📉 Visualização e Interpretação de Dados
- Exploração de dados
- Identificação de padrões
- Análise exploratória (EDA)
- Comunicação de insights

Desenvolvimento de um projeto completo de análise de dados, aplicando:
